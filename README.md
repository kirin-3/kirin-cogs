# Kirin's Red Discord Bot Cogs

A collection of custom cogs for [Red Discord Bot](https://github.com/Cog-Creators/Red-DiscordBot).

## Available Cogs

### CustomCommand
Allows users with a specific role to create a single custom command.

### CustomRoleColor
Allows administrators to assign a role to a user and lets that user customize the color, name, and icon of their assigned role.

### Patron
This cog connects to Patreon's API to fetch donation data and automatically sends award messages for both new patrons and monthly recurring donations.

### RulesAccept
Lets users accept rules via a button and modal, and assigns a role upon acceptance.

### TabooAccess
Manages access to taboo content through a button and modal system, assigning appropriate roles upon acceptance.

### Tickets
Multi-Panel Discord support ticket system with buttons!

### UnicornDocs
AI-powered documentation question and answer system for moderation team using RAG and OpenRouter.

### Unicornia
A Red bot cog that reads from Nadeko's SQLite database to provide leveling and economy features.

### UnicornModeration
Moderation cog with a papers please theme.

### UnicornSecurity
Contains moderation tools including a filter that only allows tenor GIF links in specific channels.

### VerifyUser
Allows authorized users to verify other users by granting them a verification role.

### Void
Delete messages in a channel every 24 hours.

## Installation

To install these cogs, run the following commands in Discord:

```
[p]repo add kirin-cogs https://github.com/kirin-3/kirin-cogs
[p]cog install kirin-cogs <cogname>
[p]load <cogname>
```

Replace `<cogname>` with the name of the cog you want to install.

## Requirements

- Red Discord Bot V3
- discord.py 2.0+
